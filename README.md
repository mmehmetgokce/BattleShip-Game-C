# BattleShip Game

This project is a terminal-based Battleship strategy game developed using the C programming language. The project focuses on fundamental programming logic, two-dimensional array management, and dynamic user interaction.

Bu proje, C programlama dili kullanılarak geliştirilmiş, terminal tabanlı bir Amiral Battı strateji oyunudur. Proje, temel programlama mantığı, iki boyutlu dizilerin yönetimi ve dinamik kullanıcı etkileşimi üzerine odaklanmaktadır.
